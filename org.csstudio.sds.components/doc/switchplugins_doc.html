org.csstudio.sds.cosywidgets.common.SwitchPlugins

The static initialization block of this class contains code to enumerate all the switch plugins.
The plugin identifiers are loaded into a hash map, their values set to the IConfigurationElement of the plugin. There are also two arrays, names and ids, holding the names and IDs of the plugins, respectively. Both are indexed by the index of the plugin in the selecting property of the Switch widget.
