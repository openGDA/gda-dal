org.csstudio.sds.cosywidgets.ui.internal.utils.TextPainter

This class provides functions to ease drawing of text with different alignments, possibly rotated:

  • getDelta calculates the offset of the left-center point of text relative to the given anchor.
  • drawText draws text horizontally.
  • drawRotatedText draws text, rotated for the given floating-point angle, in degrees.

The 9 possible anchor points: left-, center- and right-top, center and bottom.
