Widgets

The widgets are all formed of four classes: a model factory, a model, an editpart and a figure.
The model factory provides the model, which registers all the properties and provides functions to get their values. The editpart is the supervisor for the other three and is responsible for creating a model and a figure, as well as registering property listeners. The figure is the class that does the actual drawing.
When creating a new widget, make sure the IDs all match. A widget has these IDs in three different places: the model class, the extension point for model factories and the extension point for editparts.
