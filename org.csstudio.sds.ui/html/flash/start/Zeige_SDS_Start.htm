Wie übertrage ich eine Demo auf meinen Web-Server oder auf eine CD-ROM?

Sie müssen Ihre Demo Dateien manuell auf Ihren Web-Server hochladen, oder die Dateien in ein Verzeichnis kopieren (z.B. CD-Rom).

Diese Dateien sollten auf einen Web-Server oder in ein Verzeichnis kopiert werden:

[Schritt 2]
Nachdem Sie die Demo Dateien hochgeladen / kopiert haben, können Sie diese Demo in Ihre Internet Seiten einbinden. Als erstes kopieren Sie den folgenden Code, und fügen es zwischen den <head> und </head> Tags auf Ihrer Hompage ein:
