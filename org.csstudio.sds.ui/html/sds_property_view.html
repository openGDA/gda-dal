Widget Properties

The property view shows all properties for the widget selected in the editor. If more than one widget is selected only the properties are displayed that both have in common.
The following list contains only the properties that belongs to the most widgets. For special properties please refer to section 'Synoptic Display Studio Widgets'. Note that the properties will not be updated in the display until they are saved!

Property List

Behaviour

Action Data A set of actions can be added to all widgets. They are available on the running display in a drop down menu.
Alias If there is an alias set for a process variable it can be used in other properties of the widget to make typing easier.
Enabled Enable / Disable the widget
Permission ID xxx
Primary PV The process variable that is set as primary pv is sent to other CSS applications via object contribution (left mouse click on the widget → CSS).
Visibility Set if the widget is visible or not on the display.

Border

Border Color Color of the widget border.
Border Style Style of the border.
Border Width Width of the widget border.

Display

Background Color Background color of the widget.
Cursor Which cursor icon should be used when the cursor is above the widget.
Foreground Color Foreground color of the widget. Sometimes the font-color.
Name xxx
Tool Tip Content of the tool tip that is shown when the cursor is above the widget.

Position

Hight Hight of the widget.
Width Width of the widget.
X-Coordinate Horizontal position of the widget on the editor pane.
Y-Coordinate Vertical position of the widget on the editor pane.

Dynamisation of properties

SDS provides the possibility to dynamise each property by binding it to a process variable. The current value of the process variable controls then the behaviour of the property. The way the property changes with the value is defined in rules.
